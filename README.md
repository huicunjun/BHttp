# BHttp·极简Http库

<img src="https://v1.jinrishici.com/all.svg">

### 简介

本项目是一个不依赖Retrofit和Rxjava，但实现了retrofit的核心功能，以及rxjava的链式调用线程自动切换等。具有以下特点：

* 快：方法封装极简，链式调用一气呵成。
* 全：既可以使用单独的链式请求，也可以使用Retrofit的声明式接口写法。
* 轻：项目依赖少，仅仅依赖了Okhttp，Gson
